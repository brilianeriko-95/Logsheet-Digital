# 🏭 Plant Monitoring System

Sistem logsheet dan laporan anomali untuk plant industri.

## 🚀 Setup Cepat

### 1. Setup Google Spreadsheet
- Buat spreadsheet baru
- Buka Extensions → Apps Script
- Copy kode dari [GAS_CODE.md](./GAS_CODE.md)
- Deploy sebagai Web App
- Copy URL Web App

### 2. Edit Config di GitHub
- Buka file [`config.js`](./config.js)
- Klik ✏️ Edit (pensil)
- Ganti `YOUR_URL_HERE` dengan URL Web App Anda
- Commit changes

### 3. Aktifkan GitHub Pages
- Settings → Pages → Source: GitHub Actions
- Tunggu 1-2 menit
- Akses: `https://username.github.io/plant-monitoring`

## 📝 Cara Edit

| Aksi | Cara |
|------|------|
| Edit config | Buka `config.js` → Klik ✏️ → Edit → Commit |
| Edit tampilan | Edit `index.html` langsung di GitHub |
| Upload icon | Add file → Upload files → `icons/` |
